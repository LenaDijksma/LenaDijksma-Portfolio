# LenaDijksma-Portfolio
My personal portfolio website

## Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js
- Email: Resend
- Hosting: Render

## Running locally
1. Clone the repo
2. Run `npm install`
3. Copy `.env.example` to `.env` and fill in your keys
4. Run `node index.js`

## Environment Variables
RESEND_API_KEY=
