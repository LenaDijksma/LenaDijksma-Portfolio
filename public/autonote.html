About

Autonote Systems

Autonote and it's associated tools (newnote suite) are a set of tools that track energy en mood through a number range. It is written in bash and ran from a terminal. It creates a .md file using the path (default /Documents/Obsidian/Main/011-Autonote). It was made to track how certain lifestyle choices and changes in medication impacted my day to day life. It helped to show what helped me with increased energy and mood. The suite consists of newnote(per day file), weeklystats(average week and graph), monthlystats(same as weekly but for months).

newnote can be used multiple times a day and will append the outcome to the file although only the first use will be used by weekly/monthly stats tools.
